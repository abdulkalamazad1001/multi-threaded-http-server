About This HTTP Server

This project is a multi-threaded HTTP server developed in Python using low-level socket programming.

It is designed to handle multiple clients concurrently by using a thread pool, improving efficiency and performance under load.

Key Features

  • Concurrent Connections: Supports multiple client connections simultaneously with thread pool management.
  • Static File Serving: Serves HTML, JPEG images, and text files with correct HTTP headers.
  • JSON POST Uploads: Accepts JSON data uploads and securely saves them to the server.
  • Security: Implements path traversal protection and strict host header validation to prevent unauthorized access.
  • Connection Management: Supports persistent HTTP connections with keep-alive and connection timeouts.
  • Error Handling: Provides appropriate HTTP error responses for malformed requests and unsupported operations.
  • Comprehensive Logging: Logs all significant events including server activity, errors, and client interactions with timestamps.

Technologies Used

  • Python 3 with built-in socket and threading libraries.
  • Standard HTTP/1.1 protocol handling and compliance.
  • Basic file system operations for serving and saving files securely.

This server is primarily intended for educational purposes to illustrate core network programming and HTTP concepts.
